Скраппер веб-сайтов

recursive_scraper — это асинхронный веб-краулер, разработанный на Python с использованием библиотеки Playwright. Инструмент предназначен для извлечения и обработки HTML-контента с различных веб-сайтов. Он поддерживает обработку изображений, навигацию по ссылкам, конвертацию HTML в Markdown и хранение данных в структурированном виде.

Технологии: Python, asyncio, Playwright, BeautifulSoup, html2text, aiofiles, PyPDF2, Camelot, Logging.
