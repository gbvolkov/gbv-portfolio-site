Распознавание банковских выписок

statements_recognition — это инструмент для распознавания и обработки банковских выписок. Он позволяет автоматически извлекать важные данные из различных форматов документов (Excel, PDF) и структурировать их для дальнейшего анализа. Проект использует библиотеки Pandas и PyPDF2 для обработки данных, а также регулярные выражения для извлечения ключевой информации.

Технологии: Python, Pandas, PyPDF2, OpenCV, Регулярные выражения, Excel, PDF.
